# Project Description for Resume (Spottr)

Here are 3 options ranging from technical to product-focused:

## Option 1: Balanced (General Software Engineering)
"Developed **Spottr**, a full-stack social fitness platform using the MERN stack, featuring a custom weighted algorithm to match users based on geolocation, goals, and schedule. Engineered a RAG-powered AI Nutritionist using vector embeddings and multi-model fallback to generate scientifically grounded diet plans with per-meal regeneration. Implemented real-time chat via Socket.io and natural language processing for smart activity tracking."

## Option 2: Backend & AI Specialist
"Architected a scalable Node.js backend for a social fitness app, implementing a geospatial matchmaking engine with weighted scoring. Integrated an advanced Retrieval-Augmented Generation (RAG) system with a vector database to deliver verified nutrition advice, utilizing multi-model redundancy (Gemini/Llama) for high availability. Optimized system performance and deployed a responsive React frontend."

## Option 3: Product & Feature Focused
"Built a Tinder-style gym buddy finder connecting users via a smart compatibility algorithm (~90% match relevance). Features include a RAG-driven AI diet planner with granular regeneration, real-time messaging, and a gamified leaderboard. Utilized React, Tailwind CSS, and Framer Motion to create a high-performance, premium user interface with seamless animations."
